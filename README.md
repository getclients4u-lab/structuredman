# StructuredMan

**Navigate Divorce With Clarity**

A structured support platform for men navigating divorce — with practical tools, a clear educational roadmap, and a community of men moving forward.

## Pages

- **Home** — Hero, intro strip, four phases preview, testimonials, CTA
- **Our Mission** — Problem statement, stats, core values, founding story
- **How It Works** — Four-phase framework breakdown with deliverables
- **Curriculum** — Six core modules with lessons, tools & templates
- **Partners** — Founding partners, advisory board, partner inquiry form
- **FAQ** — Accordion FAQ, join form, contact cards

## Tech

Pure static HTML + CSS + JS. No build step.

## Deploy

```bash
vercel --prod
```
