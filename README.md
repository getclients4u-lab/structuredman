# Structured Man

**Discipline, Habits & Masculine Optimization**

A modern, SEO/AEO-optimized website for men building unbreakable discipline and structured lives.

## Features

- **Next.js 14 App Router** with static export
- **Tailwind CSS** with custom gold/slate design system
- **Full SEO/AEO Optimization**:
  - Schema.org structured data (Organization, FAQPage, Article, BreadcrumbList)
  - Dynamic sitemap.xml
  - robots.txt
  - OpenGraph & Twitter Cards
  - Canonical URLs
- **6 Content Pillars**: Morning Architecture, Physical Discipline, Mental Models, Career Weaponization, Social Calibration, Evening Recalibration
- **4 Detailed Field Guides** with rich content
- **FAQ Section** with accordion and FAQPage schema
- **Newsletter CTA** with email capture
- **Privacy Policy & Terms** pages
- **Mobile-first responsive design**
- **Admin Dashboard** (`/admin`)

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React

## Project Structure

```
structuredman/
  app/              # Next.js App Router pages
  components/       # Reusable React components
  content/          # Markdown content (future CMS)
  dist/             # Static export output (deploy target)
  public/           # Static assets
```

## Scripts

```bash
npm run dev       # Development server
npm run build     # Static export to dist/
npm start         # Serve production build
```
